# OpenCV图像处理演示程序

## 1 使用说明

下载完成后，解压到当前文件夹

解压完成后双击 OpenCV图像处理演示程序.exe 即可启动软件



## 2 软件涉及库以及库版本

本软件开发基于python3.9和pyqt5，涉及图像处理部分逻辑基于opencv-python库。
